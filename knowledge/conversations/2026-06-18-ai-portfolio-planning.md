# AI 포트폴리오 설계 대화 요약

## 핵심 아이디어

Vibe Coding Lab은 조정민의 AI 활용 능력을 보여주는 개인 포트폴리오이다.

단순히 프로젝트 결과물을 나열하는 것이 아니라, AI와 함께 아이디어를 만들고, 설계하고, 구현하고, 개선하는 과정을 보여준다.

## 주요 결정사항

- 지금은 구현보다 기획과 지식 구조화가 먼저이다.
- ChatGPT는 PM 및 기획자처럼 사용한다.
- Codex, Claude Code, Cursor는 구현 단계에서 개발자처럼 사용한다.
- 프로젝트의 핵심 자산은 코드뿐 아니라 아이디어, 프롬프트, 대화, 개발 로그이다.
- Markdown 기반 지식 베이스를 만들고, 추후 챗봇의 RAG 데이터로 활용한다.

## 프로젝트 후보

- Personal LLM Wiki
- Ask About Me Chatbot
- AI Trading Bot
- Prompt Library
- AI Build Log
- WebGPU Game

## 저장 기준

모든 대화와 프롬프트를 저장하지 않는다.

다음 기준을 만족하는 것만 저장한다.

- 3개월 뒤 다시 봐도 가치가 있는가?
- 프로젝트 방향이나 설계에 영향을 주었는가?
- 다른 사람에게 공유해도 도움이 되는가?
- 재사용 가능한 구조인가?
