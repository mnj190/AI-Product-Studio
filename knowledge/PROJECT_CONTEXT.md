# Project Context

## Project Name

Vibe Coding Lab

## Project Goal

이 프로젝트는 조정민의 AI 활용 능력을 보여주는 개인 포트폴리오 웹사이트이다.

단순한 개발자 포트폴리오가 아니라, AI를 활용해 프로젝트를 기획하고 구현하고 개선하는 과정을 보여주는 것이 핵심이다.

## Core Concept

읽는 포트폴리오가 아니라, 질문하고 탐색하는 포트폴리오를 만든다.

방문자는 웹사이트에서 조정민의 경력, 프로젝트, AI 활용 방식, 프롬프트, 개발 로그를 볼 수 있고, 챗봇에게 직접 질문할 수 있다.

## Main Message

AI를 활용해 아이디어를 빠르게 실험하고, 실제 서비스 형태로 구현하는 개발자.

## Target Audience

- 채용 담당자
- 개발자
- AI 활용에 관심 있는 사람
- 포트폴리오 방문자
- 향후 프로젝트 협업자

## Important Direction

이 사이트는 코딩 실력을 깊게 보여주는 일반 개발자 포트폴리오가 아니다.

중심은 다음이다.

- AI 활용 능력
- 바이브 코딩 과정
- 프롬프트 설계
- AI와 함께 만든 결과물
- 실제 운영 중인 프로젝트
- 개발 과정 기록

## First Version Scope

초기 버전에서는 아래 기능만 우선 구현한다.

- 개인 포트폴리오 홈
- LLM Wiki 기반 문서 페이지
- 조정민에 대해 질문할 수 있는 챗봇
- 프로젝트 목록
- Prompt Library
- Build Log

## Main Projects

### 1. Personal LLM Wiki

조정민에 대한 정보를 Markdown 문서로 구축한다.

이 문서는 나중에 LLM Wiki의 핵심 페이지이자 챗봇이 참고할 지식 계층으로 사용된다.

### 2. Ask About Me Chatbot

방문자가 조정민에 대해 질문할 수 있는 챗봇이다.

예시 질문:

- 조정민은 어떤 개발자인가요?
- 어떤 프로젝트를 했나요?
- ORIS 프로젝트는 무엇인가요?
- AI를 어떻게 활용하나요?
- 자동매매봇은 어떻게 만들었나요?

### 3. AI Trading Bot

AI와 함께 만든 자동매매봇이다.

공개 가능한 개발 과정, 기술적 의사결정, 요약 수준의 운영 회고를 포트폴리오에 반영한다.

실시간 계좌 정보, 구체적인 계좌 잔고, 구체적인 수익/손실 금액, 매수/매도 시점과 수량, API Key, 계좌번호, 투자 권유처럼 읽힐 수 있는 표현은 공개하지 않는다.

### 4. Prompt Library

프로젝트를 진행하면서 사용한 좋은 프롬프트를 저장한다.

공유 가치가 있는 프롬프트만 선별한다.

### 5. AI Build Log

AI를 활용해 프로젝트를 만든 과정을 날짜별로 기록한다.

### 6. WebGPU Game

장기 목표이다.

AI를 활용해 WebGPU 기반 게임을 만드는 프로젝트로, 포트폴리오의 고급 단계 목표이다.
