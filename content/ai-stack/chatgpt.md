# ChatGPT

## When I Use It

프로젝트 방향을 잡거나, 아이디어를 정리하거나, 정보 구조를 설계할 때 주로 사용한다.

이 프로젝트에서는 ChatGPT를 PM, 기획자, 문서 설계자처럼 사용한다.

## Strengths

- 초기 아이디어 정리
- 프로젝트 컨셉 설계
- 정보 구조 설계
- 문서 초안 작성
- 프롬프트 개선
- 작업 우선순위 정리

## Weaknesses

- 실제 코드베이스 상태를 직접 확인하지 않으면 구현 세부사항이 틀릴 수 있다.
- 최신 로컬 파일 구조를 모르면 일반적인 답변으로 흐를 수 있다.
- 구체적인 검증 없이 방향만 그럴듯하게 정리할 위험이 있다.

## Best Use Cases

- 무엇을 만들지 결정하기
- MVP 범위 정하기
- 문서 구조 만들기
- 프로젝트 이름과 메시지 정리하기
- 좋은 프롬프트 만들기
- Build Log 초안 작성하기

## Example Prompts

```text
이 프로젝트를 구현하기 전에 목표, 핵심 사용자, MVP 범위, 콘텐츠 구조를 먼저 정리해줘.
```

```text
이 대화에서 3개월 뒤에도 가치 있는 결정과 아이디어만 추려서 문서화해줘.
```
