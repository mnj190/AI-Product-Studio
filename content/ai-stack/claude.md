# Claude

## When I Use It

긴 문서, 구조화된 설명, 코드 리뷰, 복잡한 설계 비교에 사용하기 좋은 도구로 보고 있다.

## Strengths

- 긴 맥락을 읽고 정리하는 능력
- 문서 표현 다듬기
- 설계 비교
- 코드 리뷰
- 리팩토링 방향 제안

## Weaknesses

- 실제 레포 상태와 연결되지 않으면 추상적인 조언에 머물 수 있다.
- 구현 도구로 사용할 때는 현재 파일 구조와 작업 규칙을 명확히 제공해야 한다.

## Best Use Cases

- 프로젝트 문서 리뷰
- README 개선
- 긴 설계 문서 요약
- 코드 구조 리뷰
- 프롬프트 개선

## Example Prompts

```text
이 프로젝트 문서를 읽고, 포트폴리오 방문자가 이해하기 어려운 부분을 찾아줘.
```

```text
이 설계가 RAG 챗봇으로 확장하기에 적합한지 장단점을 비교해줘.
```
