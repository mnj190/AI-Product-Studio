# Interests

## Summary

현재 가장 큰 관심사는 AI를 개발 과정에 자연스럽게 통합하는 것이다.

AI를 단순히 코드 자동완성 도구로 쓰는 것이 아니라, 기획자, 설계 파트너, 코드 리뷰어, 문서화 도구, 실험 보조자로 활용하는 방법을 탐구하고 있다.

## Vibe Coding

Vibe Coding은 AI와 대화하면서 빠르게 아이디어를 구현하고 개선하는 개발 방식으로 보고 있다.

중요한 것은 속도만이 아니라, AI와 사람이 역할을 어떻게 나누고, 어떤 기록을 남기며, 결과물을 어떻게 검증하는지이다.

## LLM & RAG

개인 문서, 프로젝트 기록, 프롬프트, 개발 로그를 기반으로 답변하는 챗봇에 관심이 있다.

이 포트폴리오의 Ask About Me Chatbot도 이 관심에서 출발했다.

## AI Agent

단일 프롬프트로 끝나는 AI 사용보다, 여러 작업 단계를 나누고 반복적으로 실행하는 Agent Workflow에 관심이 있다.

예를 들어 요구사항 정리, 코드 수정, 테스트, 문서 업데이트, 로그 작성이 하나의 흐름으로 연결되는 구조를 실험하고 싶다.

## MCP

MCP는 AI가 외부 도구와 데이터를 더 자연스럽게 사용할 수 있게 만드는 연결 방식으로 보고 있다.

장기적으로는 개인 지식 베이스, GitHub, 문서, 개발 환경을 AI와 연결하는 실험을 해보고 싶다.

## AI Trading Bot

자동매매봇은 API 연동, 데이터 저장, 전략 테스트, 로그 분석, 운영 안정성까지 포함하는 좋은 실험 주제이다.

다만 투자 권유가 아니라 개발 실험과 운영 기록의 관점으로 다룬다.

## WebGPU Game

장기 목표로 AI를 활용해 WebGPU 기반 게임을 만드는 실험을 해보고 싶다.

포트폴리오에서 기술적으로 더 도전적인 프로젝트가 될 수 있다.
