# Daily Work Log Prompt

AI와 함께 한 작업을 매일 아침 공유용 기록으로 정리하기 위한 프롬프트다.

## Prompt

```text
어제 한 작업 또는 아직 기록되지 않은 모든 작업을 확인해서 공유용 Daily Digest로 정리해줘.

목적은 AI로 어떤 식으로, 어떻게, 얼마나 작업했는지 보여주는 거야.

다음 내용을 포함해줘.

1. 어제 만든 결과물 요약
2. AI를 사용한 방식
3. 중요한 의사결정
4. 바이브 코딩 초보자가 참고할 만한 요청/프롬프트
5. 검증한 내용
6. 아직 남은 일

기존 Build Log가 있으면 하루 단위로 묶고, 빠진 작업이 있으면 먼저 작업 로그를 추가해줘.
민감한 정보나 비공개 정보는 공개 로그에 넣지 말고 요약해줘.
```

## When To Use

- 매일 아침 작업 기록을 남길 때
- 여러 개의 작은 작업 로그를 하루 요약으로 묶고 싶을 때
- AI 협업 과정을 포트폴리오 콘텐츠로 바꾸고 싶을 때
- 바이브 코딩을 처음 하는 사람에게 과정을 보여주고 싶을 때

## Why It Works

이 프롬프트는 단순히 “무엇을 했는지”가 아니라 “AI를 어떻게 사용했는지”를 기록하게 만든다.

결과물, 의사결정, 프롬프트, 검증을 함께 남기기 때문에 나중에 포트폴리오, 블로그, 회고, 강의 자료로 재사용하기 좋다.

## Reusable Pattern

```text
[기간] 동안 한 작업 또는 아직 기록되지 않은 작업을 확인해서 [공유 목적]에 맞는 기록으로 정리해줘.

포함할 내용:
- 만든 결과
- AI를 사용한 방식
- 중요한 요청/프롬프트
- 의사결정
- 검증
- 다음 작업

민감한 정보는 제외하고, 처음 보는 사람도 흐름을 이해할 수 있게 써줘.
```
