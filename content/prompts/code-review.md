# Code Review Prompt

## Purpose

AI가 생성하거나 수정한 코드를 검토할 때 사용하는 프롬프트이다.

기능이 동작하는지만 보는 것이 아니라, 프로젝트 방향과 문서 기준에 맞는지도 함께 확인한다.

## Used In

- Next.js MVP 구현 후 점검
- Markdown 렌더러 검토
- LLM Wiki 기반 질문형 인터페이스 구현 전 구조 검토
- 리팩토링 전 위험 요소 확인

## AI Tool

- Codex
- Claude
- ChatGPT

## Prompt

```text
다음 코드를 리뷰해줘.

리뷰 기준은 다음이야.

1. 요구사항을 실제로 만족하는가?
2. 불필요하게 복잡하지 않은가?
3. 이후 기능 확장에 방해되는 구조는 없는가?
4. 보안상 위험한 정보 노출 가능성은 없는가?
5. 이 프로젝트의 문서/콘텐츠 중심 방향과 맞는가?
6. 수정이 필요하다면 우선순위별로 알려줘.

단순 스타일 지적보다 실제 유지보수와 확장성에 중요한 문제를 먼저 봐줘.
```

## Result

AI가 만든 결과물을 그대로 믿지 않고, 개발자가 책임지고 검증하는 흐름을 만들 수 있다.

## Improved Version

```text
너는 이 프로젝트의 코드 리뷰어야.

이 프로젝트의 목적은 AI 활용 과정을 보여주는 Markdown 기반 포트폴리오야.
코드를 리뷰할 때 기능 동작뿐 아니라 다음도 함께 확인해줘.

- 콘텐츠 추가가 쉬운가?
- 문서와 구현이 어긋나지 않는가?
- LLM Wiki 기반 질문형 인터페이스 확장에 방해되지 않는가?
- 민감한 정보가 공개될 위험은 없는가?
- 지금 단계에서 과한 구현은 없는가?

발견한 문제는 Critical / Important / Nice to Have로 나눠줘.
```

## Notes

AI 코드 리뷰는 정답을 받는 과정이 아니라, 놓친 관점을 찾는 과정으로 사용한다.
