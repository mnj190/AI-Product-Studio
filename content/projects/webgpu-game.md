# WebGPU Game

## Summary

AI를 활용해 WebGPU 기반 게임을 만드는 장기 실험 프로젝트이다.

현재는 구현 전 아이디어 단계이며, AI Product Studio의 고급 단계 목표로 둔다.

## Why I Built This

WebGPU Game은 단순 웹 포트폴리오보다 기술적으로 더 도전적인 프로젝트가 될 수 있다.

그래픽스, 게임 루프, 상태 관리, 입력 처리, 성능 최적화, 에셋 관리 등 다양한 주제를 다루기 때문이다.

AI를 활용해 생소한 기술 영역을 어떻게 학습하고 실험하는지 보여주기에 좋은 장기 프로젝트이다.

## AI Usage

AI는 다음에 활용할 계획이다.

- WebGPU 개념 학습
- 렌더링 파이프라인 이해
- 작은 데모 설계
- shader 코드 실험
- 게임 루프 구조 설계
- 디버깅과 성능 개선
- 개발 로그 정리

## Main Features

초기 아이디어:

- 작은 WebGPU 데모
- 캐릭터 또는 오브젝트 렌더링
- 키보드 입력 처리
- 간단한 게임 루프
- 성능 측정 로그
- AI와 함께 만든 shader 실험 기록

## Tech Stack

후보:

- TypeScript
- WebGPU
- Canvas
- WGSL
- Next.js 또는 별도 Vite 프로젝트

## Prompt Examples

- WebGPU를 처음 학습하는 개발자를 위한 최소 데모 로드맵을 만들어줘.
- 이 WGSL shader가 어떤 역할을 하는지 설명해줘.
- WebGPU 게임 루프를 작은 단위로 구현하는 순서를 정리해줘.

## Build Process

1. WebGPU 기본 개념을 Wiki로 정리한다.
2. 최소 렌더링 데모를 만든다.
3. 입력 처리와 게임 루프를 붙인다.
4. AI를 활용한 shader 실험을 기록한다.
5. 독립 레포로 분리할지 결정한다.

## Result

아직 구현 전 단계이다.

현재는 장기 목표로 관리하며, AI Product Studio 안에서는 프로젝트 소개, 학습 경로, 관련 Wiki를 먼저 정리한다.

구현이 시작되기 전까지는 별도 레포를 만들지 않고, 공개 가능한 계획과 학습 기록만 이 포트폴리오에 남긴다.
