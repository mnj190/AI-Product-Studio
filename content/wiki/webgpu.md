# WebGPU

## Summary

WebGPU는 웹에서 GPU를 활용한 그래픽스와 계산 작업을 할 수 있게 하는 기술이다.

AI Product Studio에서는 장기 목표인 WebGPU Game 프로젝트의 기반 기술로 관심을 두고 있다.

## Why It Matters

WebGPU는 일반적인 웹 UI를 넘어 그래픽스, 시뮬레이션, 게임, 고성능 렌더링 실험으로 확장할 수 있는 기술이다.

포트폴리오 관점에서는 AI를 활용해 낯선 고급 기술을 학습하고 실제 데모로 만드는 과정을 보여줄 수 있다.

## Planned Learning Path

초기 학습 순서:

1. Canvas와 WebGPU 기본 구조 이해
2. Device, adapter, command encoder 개념 정리
3. 삼각형 렌더링
4. WGSL shader 이해
5. 애니메이션 루프
6. 입력 처리
7. 작은 게임 데모

## AI Usage

AI는 다음에 활용할 수 있다.

- 개념 설명
- 코드 예제 분석
- shader 설명
- 디버깅
- 작은 실험 설계
- Build Log 작성
